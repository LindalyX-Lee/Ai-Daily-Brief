OPENCLAW UPDATE CHECK
- Current version: OpenClaw 2026.3.8 (3caab92)
- Latest version: v2026.3.24
- Recommend update? Yes — 当前版本明显落后；v2026.3.24 直接补了 OpenAI 兼容接口、tool 可见性、容器 CLI、技能安装 UX 与多通道稳定性修复，都是日常 builder workflow 高频面。建议手动受控升级，但不要自动升。
- Gateway/OpenAI compatibility 新增 /v1/models 与 /v1/embeddings，并让显式 model override 能穿透 chat/responses，兼容更多 client 与 RAG 流程
- Agents/tools 现在会显示当前 agent 此刻真正可用的 tools，Control UI 也新增 Available Right Now 区域，减少“以为能用其实不能用”
- Skills 元数据支持一键安装 recipe，Control UI 也新增 Ready / Needs Setup / Disabled 状态筛选和更完整的 setup 引导
- CLI 新增 --container 与 OPENCLAW_CONTAINER，可直接对运行中的 Docker/Podman 容器执行 openclaw 命令
- 修复 outbound media/fs policy、gateway restart 唤醒、channel sequential startup 等稳定性问题，减少多通道和重启后的隐性故障
