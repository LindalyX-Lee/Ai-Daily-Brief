OPENCLAW UPDATE CHECK
- Current version: OpenClaw 2026.4.15 (041266a)
- Latest version: v2026.4.16
- Anthropic 默认模型选择与 Claude CLI defaults 对齐到 Opus 4.7，说明 agent runtime 正在更积极偏向长任务和高可靠 coding workflow。
- Google plugin 新增 Gemini TTS，包含 voice 选择、WAV reply output 和 telephony PCM，意味着 multimodal output 已从展示能力走向可嵌入真实产品流。

- memory-lancedb 支持云存储，durable memory index 不再被本地磁盘绑定，agent memory 开始更像 production infra。
- 新增 experimental.localModelLean，可为弱本地模型移除 browser、cron、message 等重工具，说明 agent stack 开始认真做低配部署优化。
- 修复 built-in tool name collision 的本地 MEDIA trust 漏洞和 webchat audio path containment，这波有明确安全收益。
- Recommend update? Yes, 这次不只是小修小补，覆盖默认模型、TTS、多 provider 运维可视化、memory infra 和安全修复，手动受控升级值得排期。
